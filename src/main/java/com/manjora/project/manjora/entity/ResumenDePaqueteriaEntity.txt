El paquete "Entity" tiene como objetivo
guardar las clases que contienen los metodos para
la creacion directa de las tablas de entidad en la base de
datos para su facil actualizacion.

Ejemplo:

1.-Generas la clase de Usuario
2.-Defines sus atributos como si creases una tabla en la BD
3.-Spring propiamente generara las tablas en la base de datos
4.-Si se genera algun cambio en el codigo, spring actualizara automaticamente en la BD
